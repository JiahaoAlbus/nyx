# NYX Reference UI (Week 5 Preview)

## Purpose
Provide a thin, text-only evidence viewer that integrates with the backend evidence service.

## Scope
- Static HTML pages.
- Evidence display and export via backend API.

## Non-Scope
- No authentication.
- No protocol semantics or fee logic.
- No live system data.

## Run (Dev)
Start the backend and open the UI:
- http://localhost:8080/home

## Evidence
The evidence page renders fields verbatim from the backend and provides a deterministic export.
