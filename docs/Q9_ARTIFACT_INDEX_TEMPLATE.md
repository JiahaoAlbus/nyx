# Q9 Artifact Index Template

Purpose
- Provide a consistent index format for Q9 deliverables and evidence artifacts.

Scope
- Q9 docs, backend outputs, iOS app deliverables, and verification artifacts.

Template Columns
- Path
- Artifact Type (Docs / Code / Evidence / Ops)
- Purpose
- Normative (Yes/No)
- Verification Command

Rules (MUST / MUST NOT)
- The index MUST list all Q9 deliverables and their verification commands.
- Entries MUST be written in English and must be factual.
- The index MUST NOT claim live mainnet status.

Change Control
- Updates to the index must preserve existing entries and add new items additively.
